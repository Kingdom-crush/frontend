# HDL 规则检查前端演示

这是一个纯静态前端演示稿，用于讨论 Verilog/VHDL 静态规则检查平台的页面流程和可行性边界。

## 打开方式

直接用浏览器打开 `index.html` 即可，不需要安装依赖，不连接真实引擎。

## 覆盖范围

- 新建检查项目
- 被测代码管理
- 规则集管理与规则等级调整
- Windows / Linux / 麒麟 ARM64 的引擎命令预览
- 检查执行进度演示
- 结果统计、热点图、详情表、人工审计
- 规则帮助栏
- 报告模板管理
- RTL 电路图查看入口
- 系统适配与技术边界说明

## 演示边界

- 当前版本不调用 `spec-hdl` 或 `vhdl-checker.jar`。
- 当前版本不读写真正的 `toif.db`。
- 报告、软件锁、RTL 图均为前端占位，用于讨论后端接口。
